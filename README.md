# Genesis 1 Hebrew Study

A small static Bible study app for exploring Hebrew words in Genesis 1.

Open `index.html` in a browser. Click a Hebrew word to see:

- English gloss
- Transliteration
- Hebrew root
- Form and grammar
- Context notes
- A study prompt

The app currently includes Genesis 1:1-5 with word-level notes. To add more verses, extend the `verses` array in `script.js`.
